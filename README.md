# Registro Visitatori

Progetto realizzato per il corso di **Laboratorio di Interfaccia Uomo-Macchina**.

L’applicazione consente la gestione di un registro visitatori tramite un’interfaccia web, con sistema di autenticazione.

---

## Tecnologie utilizzate

- ASP.NET Core MVC
- C#
- Entity Framework Core
- Database SQLite
- HTML / CSS / Bootstrap

---

## Struttura del progetto

Il progetto segue il pattern **MVC (Model-View-Controller)**:

- **Models**: modelli dei dati
- **Views**: interfaccia utente
- **Controllers**: logica applicativa
- **Data / Migrations**: gestione del database
- **wwwroot**: risorse statiche
